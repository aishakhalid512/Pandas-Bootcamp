Data Types¶

Data Types are foundational in computer Programming because they help the computer in determining how a value will be stored in memory and what operations can and cannot be performed on that given value.

Each Object in Python will be associated with a Type. It could either be a Fundamental datatype, as listed below.

int , float , bool , str , tuple , list , set , dict

There is also a Data type called "None"

None represents the absence of a value.

And there could also be specialized Data Types.

Each Object in Python will be associated with a Type. The Type will determine what properties and behaviours apply to that object.

Floating Point Numbers are approximations¶

Computers store Floating Point Numbers as binary (base 2) fractions.

For example :

  • Lets say we have 0.1
  • That is 1/10
In [56]:
1 / 10
Out[56]:
0.1

This is a base 10 Number system

But , to a computer everything is binary. A computer needs to break this down and map it to a base 2 representation. Its impossible to represent 0.1 in binary, so it stores something like

In [59]:
#expected Output is 0.3
#But we get a wierd looking output
#This is something to be aware of if you find this result surprising
0.1 + 0.2
Out[59]:
