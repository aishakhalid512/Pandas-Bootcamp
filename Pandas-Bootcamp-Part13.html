The entire JSON object is wrapped in curly braces , which reminds us of Python Dictionary or Javascript objects.

The other important structures in JSON are Ordered lists. They are quite similar to Python Lists. They just contain sequence of values separated by commas.

What we did?¶

  • We started with the Python object : "A Dictionary"
  • We serialized it using the "pickle" module in Python creating "my_stock" file in our Local Directory
  • Which is not very readable
  • Look at the serialized object below, how it looks
  • Then we read it back in
  • We loaded it using the "pickle" module
  • We exported using the "dump" function
  • When we read it back in , we use the "load" function ............................................................

This is how we went from Python Dictionary to bytes and back to Python Dictionary
